# Практична робота №1  
## Основи веб-програмування  
### Варіант 6 - Розподільча підстанція РП-10 (35/10 кВ)

## Опис роботи

У межах практичної роботи було розроблено односторінковий інформаційний веб-сайт енергетичного об’єкта - розподільчої підстанції РП-10 напругою 35/10 кВ.  

Сайт містить структурований опис об’єкта, таблицю технічних параметрів, зображення та контактну інформацію. Під час реалізації використано HTML5, CSS3 та фреймворк Bootstrap 5.

## Мета роботи

Метою роботи є закріплення навичок створення адаптивних веб-сторінок із використанням семантичної розмітки HTML5, стилізації за допомогою CSS3 та компонентів Bootstrap.

## Використані технології

У проєкті застосовано HTML5 для побудови семантичної структури сторінки та логічної організації контенту. CSS3 використано для створення власних стилів оформлення, реалізації hover-ефектів та забезпечення адаптивності інтерфейсу.  

Для спрощення верстки та реалізації адаптивного меню, таблиць і сіткової структури сторінки використано фреймворк Bootstrap 5.3.

## Як запустити проєкт

1. Клонувати репозиторій:
```
git clone https://github.com/DiachenkoDenis/Web/tree/main/Lab1
```

2. Відкрити файл `index.html` у браузері.
