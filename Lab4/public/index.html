Практична робота №4

REST API для моніторингу ТЕЦ

Об’єкт автоматизації

Варіант 6 - теплоелектроцентраль (ТЕЦ). API дозволяє отримувати, створювати, оновлювати та видаляти дані про когенераційні установки, а також переглядати дані про паливо та викиди.

Доступні endpoints

GET /api/chp-plants GET /api/chp-plants/1 GET /api/chp-plants/1/fuel GET /api/chp-plants/1/emissions
POST /api/chp-plants
POST /api/chp-plants/1/readings
PUT /api/chp-plants/1
PATCH /api/chp-plants/1
DELETE /api/chp-plants/1

Підтримувані параметри

  • name — назва ТЕЦ
  • electricPower — електрична потужність
  • thermalPower — теплова потужність
  • fuelType — тип палива
  • fuelConsumption — витрата палива
  • efficiency — ККД
  • emissions — викиди CO₂
  • steamPressure — тиск пари
  • status — стан роботи

Приклади фільтрації

За статусом:

/api/chp-plants?status=active

За типом палива:

/api/chp-plants?fuelType=gas

Пошук по назві:

/api/chp-plants?search=Київ

Сортування:

/api/chp-plants?sortBy=emissions&order=desc
